# 소개
HuggingFace BERT를 활용한 한글 리뷰 감정 분석

# 실행법
코랩과 구글 드라이브를 마운트합니다.

마운트 된 구글드라이브에 ko_data.csv 파일 삽입합니다.

테스트 결과 파일을 csv로 import 하기 위해 구글 드라이브에 경로 및 파일명을 설정합니다. 

실행하시면 됩니다.
